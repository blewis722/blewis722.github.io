Project Overview

This application will be used to promote a wedding photography business

The inteded users of this site are users that are looking for a wedding photographer

The content will be an photogallary to show the client's previous work. Also there will be content about the client so the users can get to know the client. There will be content about the client's previous customers and their testimonies. There will also be a way for the users to submit information, so the client can see when a user is interested.

Wireframe

Page Design

Home Page:

This page is designed to welcome users to the site.The conten of the page will be a few images and the menu items. There will be no need to ask users for information on this page. No actions will happen on this page.

About:

This page is intended to give the clients more information about this company. They will see how the client got started in the business and why they love wedding photography. There will be no need to ask users for information on this page. No actions will happen on this page.

Experience:

This page is about the client's previous customers and their testimonies. The users can see other people's reviews about their services with this company.

Portfolio:

This page will show the client's work. The users will be able to see different images and previous work the client has done. There will be images of engagement photoshoots and wedding photoshoots. There will be no need to ask users for information on this page. They will be able to click through different images.

Contact:

This page is intended for users interested in the client's service. This page will request their email and phone number. Also this page will ask them what kind of photoshoot they are interested in.

Dynamic Functionality

The users will be able to look through the photo gallary.
